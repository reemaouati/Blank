# Blank
Opens links in a new background tab right next to your current one.

## Install
Import script from URL:
```
https://raw.githubusercontent.com/reemaouati/Blank/refs/heads/main/blank.js
```

## Browser
- Tested on [Via](https://viayoo.com/en/)
- Tested on [Soul](https://play.google.com/store/apps/details?id=com.mycompany.app.soulbrowser&hl=en)
